# Practices_Astroinformatics
Repository of graded practices for Astroinformatics I at Universidad de Antofagasta.

The TESS light curves were obtained from the MAST Archive as FITS files and converted into CSV files using the astronomical table manager TOPCAT, which also provided a preliminary visualization of the light curves.

The final data analysis was performed in Python using the NumPy, pandas, and Astropy libraries. The data were analyzed using basic statistical methods, outlier detection, and a subsequent LombScargle period analysis.

The final results were plotted and saved in the Practice_3/plot folder.
